About Code Diagramerz

A powerful tool for generating UML class diagrams from Java code

About the Project

Code Diagramerz is an intuitive web-based tool that automatically generates UML class diagrams from Java source code. It helps developers visualize class relationships and code structure efficiently.

Features

  • Upload single or multiple Java files
  • Upload entire folders containing Java files
  • Built-in code editor with syntax highlighting
  • Real-time diagram generation
  • View both diagram and Mermaid syntax
